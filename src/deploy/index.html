JSVecX by raz0red is a JavaScript port of the VecX Vectrex emulator. The original C version was developed by Valavan Manohararajah. For status updates, known issues, or to provide feedback please visit the JSVecX forum thread.

The rendering in this port relies on the HTML5 Canvas element (no flash is required). The port is pretty CPU intensive, so you will need a modern computer with Google Chrome, Firefox (4.0), Opera, or a recent version of Safari.
